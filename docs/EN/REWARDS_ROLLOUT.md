# Proof of Peace Rewards — Rollout (EN)

- Rewards are **not trade cashback**.
- Eligibility is generated only by `ArafEscrow` terminal outcome.
- Backend is mirror-only; not authority.
- Admin and sponsor cannot choose recipients.
- `paymentRiskLevel` is not a reward multiplier.
- Auto-release, burn, mutual cancel, disputed release are zero-weight in MVP.
- Tier 0 is not reward eligible in MVP.
- `rewardBps` starts at 4000 and is bounded to 4000–7000.
